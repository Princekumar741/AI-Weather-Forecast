# 🌤️ AI Weather Forecast Project

This is a web-based weather forecast application built using Python, Flask and Machine Learning.

## 🚀 Features
- Real-time weather data
- AI-based weather prediction
- Temperature, humidity, pressure forecast
- Simple and responsive UI

## 🛠️ Technologies Used
- Python
- Flask
- HTML, CSS, JavaScript
- Machine Learning (Scikit-learn)
- OpenWeather API

## ⚙️ How to Run
1. Clone the repository
2. Install requirements using:
   pip install -r requirements.txt
3. Run the app:
   python app.py
4. Open browser and go to:
   http://127.0.0.1:5000/

## 🔑 API Key Setup
Create a `.env` file and add:
API_KEY=your_api_key_here
